# react-native-adgeist

Adtech platform to serve advertisements to publishers app

## Installation

```sh
npm install react-native-adgeist
```

## Usage


```js
import { BannerAd } from 'react-native-adgeist';

// ...

<BannerAd
    dataPublisherId={PUBLISHER_ID}
    dataAdSlot={ADSPACE_ID}
    width={400}
    height={300}
/>
```
